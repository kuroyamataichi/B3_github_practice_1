# 春日語生成プログラム
**春日語辞典**


## 概要
オブです～．このプログラムは，常人で生きるのが厳な世の中を，春日語を用いて生き抜こうという目的で作られている．これを使用すれば，VO5.春日のありがた男な言葉はまさに，副作用なし，体に優しい春日ドーピング．それではまた，夢でお会いしやしょう．アディオス．


## 特徴
*手軽に春日語をTのしみ男


## 動作環境
*python 3.6以降


## インストール方法
1.リポジトリのクローン
```bash
git clone https://github.com/kuroyamataichi/B3_github_practice_1
```
2.ディレクトリに移動
```bash
cd B3_github_practice_1
```


## 使い方
以下のコマンドを実行
```bash
python kasugago.py
```
